05 — Veille technologique

Veille

Sujet de veille · 2025

Fin du support Windows 10 & migration vers Windows 11

Le 14 octobre 2025, Microsoft a mis fin au support étendu de Windows 10. Plus aucun patch de sécurité n'est publié, exposant les machines non migrées à des vulnérabilités critiques. Pour les entreprises gérant un grand parc informatique, cette échéance représente un enjeu majeur.

  • Windows 10 EOS
  • Windows 11
  • Cybersécurité
  • Gestion de parc
  • Migration

Contexte

Pourquoi c'est critique ?

Windows 10 représentait encore plus de 60 % des postes Windows en entreprise à l'approche de la date butoir. Une fois le support terminé, Microsoft ne corrige plus les failles découvertes : chaque nouvelle CVE publiée devient une porte d'entrée potentielle pour les attaquants, sans remède officiel.

En entreprise, un seul poste non mis à jour peut suffire à compromettre tout le réseau via un ransomware ou un mouvement latéral.

Obstacles techniques

Les prérequis de Windows 11

Windows 11 impose des exigences matérielles strictes absentes sur beaucoup de postes existants :

  • TPM 2.0 — puce de sécurité requise, absente sur les machines avant 2017 environ
  • Secure Boot — doit être activé dans le BIOS/UEFI
  • Processeur compatible — Intel 8e génération ou Ryzen 2000 minimum
  • 4 Go RAM / 64 Go stockage — minimum officiels

Dans les grandes entreprises, une part non négligeable du parc peut être matériellement incompatible, forçant un renouvellement coûteux.

Impact en milieu professionnel

Ce que ça change pour les techniciens

Chez SIPCO, la fin du support Windows 10 a concrètement impacté le quotidien : les clients sont informés du risque, et la migration devient une mission récurrente. Plusieurs actions sont nécessaires :

  • Inventaire du parc — identifier les postes sous Windows 10 via l'outil de supervision (Datto RMM)
  • Test de compatibilité — vérifier TPM, Secure Boot et CPU poste par poste
  • Planification des mises à niveau — upgrade in-place pour les machines compatibles
  • Renouvellement matériel — remplacement des postes trop anciens
  • Formation des utilisateurs — prise en main de la nouvelle interface Windows 11

Alternatives & solutions

Que faire pour les postes incompatibles ?

  • ESU (Extended Security Updates) — Microsoft propose une année supplémentaire de correctifs payante pour les entreprises qui ne peuvent pas migrer immédiatement
  • Bypass TPM — contournement non officiel via registre, déconseillé en production
  • Remplacement du parc — solution pérenne mais coûteuse, à planifier sur plusieurs années
  • Isolation réseau — segmenter les postes non migrables en VLAN pour limiter l'exposition

Sources utilisées

  • Microsoft Learn — prérequis officiels Windows 11
  • CERT-FR — alertes & bulletins de sécurité Microsoft
  • Le Monde Informatique — actualité migration parc
  • Next (ex-Next INpact) — analyses techniques
  • Zataz — vulnérabilités & cyberattaques

Synthèse · Octobre 2025

Ce qu'il faut retenir

La fin du support Windows 10 n'est pas un simple changement de version : c'est une rupture de la chaîne de sécurité. Chaque jour supplémentaire passé sous Windows 10 sans correctifs augmente la surface d'attaque.

Pour un prestataire informatique comme SIPCO, cela se traduit par une vague de demandes de migration à gérer en priorité, en commençant par les postes exposés sur Internet ou connectés à des données sensibles.

  • EOS 14/10/2025
  • CVE sans patch
  • TPM 2.0
  • Upgrade in-place
